# 📚 Projeto: Notas Alunos

## 📝 Descrição

O **Notas Alunos** é um projeto simples em **Python** que permite gerenciar informações de alunos e calcular suas médias de notas. O sistema utiliza um **dicionário de alunos** com ID, nome, idade, curso e notas, e permite consultar a situação acadêmica de cada estudante.

Este projeto é ideal para **iniciantes em Python** que desejam praticar manipulação de dicionários, estruturas de decisão (`if/else`) e operações matemáticas básicas.

---

## ⚡ Funcionalidades

- Consultar dados do aluno pelo **ID**  
- Calcular a **média das notas** do aluno automaticamente  
- Determinar a **situação do aluno** com base na média:  
  - Média ≥ 7 → **Aprovado** ✅  
  - Média ≥ 5 e < 7 → **Recuperação** ⚠️  
  - Média < 5 → **Reprovado** ❌  
- Exibir informações completas do aluno:  
  - Nome  
  - Idade  
  - Curso  
  - Notas  
  - Média  
  - Situação  

---

## 🛠 Tecnologias Utilizadas

- Python 3  
- Estruturas de dados: dicionários e listas  
- Estruturas de decisão: `if/elif/else`
